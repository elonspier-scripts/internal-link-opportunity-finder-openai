# 🔗 Semantic Link Finder (OpenAI Edition)

De **Semantic Link Finder** is een krachtige SEO-tool gebouwd met Python en Streamlit. De tool gebruikt OpenAI's `text-embedding-3-small` model om de semantische relevantie tussen verschillende pagina's op een website te berekenen. Hiermee vind je in enkele seconden de beste interne linkkansen voor je focus URL's.

## 🚀 Functies

* **Semantische Analyse:** Vergelijkt content op basis van betekenis in plaats van alleen exacte keyword-overlap.
* **URL Cleaning:** Verwerkt URL-slugs automatisch naar leesbare termen voor betere context.
* **Topical Categorization:** Extraheert automatisch de belangrijkste onderwerpen per pagina.
* **Visuele Scores:** Resultaten worden gepresenteerd met een intuïtief kleurverloop (Rood/Geel/Groen).
* **Excel Export:** Download alle gevonden linkkansen direct in een professioneel opgemaakt Excel-bestand.
